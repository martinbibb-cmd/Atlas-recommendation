# Educational Print Equivalents

## Purpose

A print equivalent is a static, print-safe explanation of an educational animation or interactive visual.

It exists to preserve the same concept meaning when motion is not available.

## Core rules

- A print equivalent is **not** a screenshot of animation frames.
- A print equivalent must preserve the same concept meaning as the source animation.
- A print equivalent must stay low-ink and readable in print-first contexts.
- Animation remains available as a QR/deep-dive path for richer motion detail.
- The print version must explain the core idea without requiring motion.

## Scope boundary

- Print equivalents do not add recommendation logic.
- Print equivalents do not introduce new recommendation claims.
- Print equivalents should summarise and sequence the same educational concept already present in the source asset.
