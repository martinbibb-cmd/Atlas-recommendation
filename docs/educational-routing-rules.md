# Educational Routing Rules (Atlas Accessibility Library)

## Purpose

Educational routing is a deterministic explanation-selection layer for welcome packs, portal journeys, print packs, and engineer appendices.

## Guardrails

- Routing is explanation selection only.
- Routing does not recommend systems.
- Routing does not rank scenarios.
- Routing does not create new physics.
- Routing may select explanations, but must never alter Atlas recommendations or engine outputs.

## Deterministic accountability rule

Every selected asset must answer:

> Why is this in this customer's pack?

The selector emits explicit inclusion and omission reasons to keep that traceability explicit across routing and pack composition.
